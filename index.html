Performance Considerations

JavaScript

  • Single Threaded
  • Scripting Language (slower)

( C Performance is 1.0 )

JavaScript & WebGL

  • Keeping 60FPS is important.
  • Use requestAnimationFrame NOT setInterval.
  • RequestAnimationFrame runs at 60Hz.
  • Your render routine has ~16ms to run.

Non-WebGL GPU Accelerated Graphics

Composite Only Style Props

Composite only style properties when changed do not require style or layout calculations, they also do not force a repaint. The only proessing is the compositing of layers in the GPU.

  • Opacity
  • Transform
